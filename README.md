#MyNotesApp
MyNotesApp is a MERN-based web application that allows users to securely create, manage, and organize personal notes.
The application includes user authentication and authorization, ensuring that each user can access only their own notes.

#Features
User registration and login
Secure authentication & authorization (JWT-based)
Create, Read, Update, and Delete (CRUD) notes
Notes are private and user-specific
Separate collections for Users and Notes
Responsive and user-friendly interface
